# Product Overview

SigVisualizer is a PyQt5 GUI application for real-time visualization of electroencephalogram (EEG) signals streamed from Lab Streaming Layer (LSL). 

## Core Features
- Real-time signal visualization with customizable time windows
- Multi-stream support with expandable tree view for stream/channel selection
- Interactive channel toggling and visibility controls
- Marker data display in tabular format
- Auto-refresh capabilities with manual override options
- Panel toggling for optimized screen real estate

## Target Use Case
Designed for neuroscience researchers and practitioners who need to monitor live EEG data streams in laboratory or clinical settings. The application provides immediate visual feedback on signal quality and allows for real-time monitoring of multiple data channels simultaneously.
